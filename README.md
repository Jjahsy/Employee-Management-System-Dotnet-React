# Employee Management System

A full-stack CRUD application built using .NET Core 10 Web API and React (Vite) with Material UI.

## Tech Stack
- **Backend:** .NET Core 10 Web API, Entity Framework Core, SQL Server Express
- **Frontend:** React (Vite), Material UI (MUI), Axios

## How to Run Project

### Backend Setup:
1. Navigate to `EmployeeManagementAPI` folder.
2. Run database update command: `dotnet ef database update`
3. Run the API: `dotnet run` (Runs on http://localhost:5036)

### Frontend Setup:
1. Navigate to `employee-ui` folder.
2. Install packages: `npm install`
3. Run project: `npm run dev` (Runs on http://localhost:5173)
